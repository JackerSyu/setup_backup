/* vim 的常用指令 */

:wq 存nx_
:version 查看vimrc位置及版本
:echo $home 查vim位置

:new filename 水平分割
:vnew filename 垂直分割
:tabe filename 新增分

按住Ctrl + W，然后再加上h, j, k, l，分别表示向左、下、上、右移动窗口
Ctrl + w + h：向左移动窗口
Ctrl + w + j： 向下移动窗口
Ctrl + w + j： 向上移动窗口
Ctrl + w + l： 向右移动窗口

Ctrl + w + w：这个命令会在所有窗口中循环移动
Ctrl + w + t：移动到最左上角的窗口
Ctrl + w + b：移动到最右下角的窗口
Ctrl + w + p：移动到前一个访问的窗口

gt 切Q
u 回蜕幼

y }u反白
yy }u行
p N上
dd h除
o 下行插入
v 反白

gg 第一行
G  最後一行

A 跳至行最後面KM入模式
行 → G	移拥降 n 行
0	移拥皆行最前面
$	移拥皆行最後面
字 → [Space]	向右移 n 字元
行 → [Enter]	向下移 n 行

>> or <<   涡锌s排
V 反白 < or > 多行s排
